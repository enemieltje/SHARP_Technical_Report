\section{Reference Mission Profile and Design Assumptions}%
\label{sec:mission_scenario}
At the time of this project, the UAV platform developed by Wren Aerospace was still under active development and no final operational mission profile had yet been defined. As a result, a representative reference mission profile was established to support the design, simulation and validation of the SHARP-PS power generation and feed system.

The purpose of this reference mission profile is not to define the final operational use-case of the UAV, but rather to provide a consistent set of engineering assumptions from which system requirements and design parameters can be derived. The assumptions presented in this section are based on preliminary performance targets provided by Wren Aerospace and discussions with project stakeholders.

The target platform is a fixed-wing electrically propelled UAV intended for long-endurance operation. The primary design objective is to demonstrate the feasibility of extending UAV mission duration using a hydrogen fuel cell based power generation system integrated with Solid Flow’s cool gas generator and Stellar Space Industries’ flow control unit.
